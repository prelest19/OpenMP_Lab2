Лабораторная работа №2: Параллельное умножение матриц с OpenMP

1. Цель работы
   
Модифицировать программу из лабораторной работы №1 для параллельной работы с использованием технологии OpenMP. Провести серию экспериментов с разным количеством потоков (1, 2, 4, 8), разными размерами матриц (200, 400, 800, 1200, 1600, 2000) и разным количеством вычислительных ядер. Оценить ускорение и эффективность параллелизации.
