Лабораторная работа №2: Параллельное умножение матриц с OpenMP

Описание
Программа умножает квадратные матрицы, используя технологию OpenMP для распараллеливания вычислений.

Требования
- Компилятор с поддержкой OpenMP (g++ 4.9 или новее)
- Git Bash или любой терминал Linux/macOS

Структура проекта
OpenMP_Lab2/
├── data/ # Папка с файлами матриц (генерируется)
│ └── .gitkeep # Пустой файл, чтобы папка сохранилась в Git
├── main.cpp # Основная программа с OpenMP
├── generate.cpp # Генератор тестовых матриц
├── .gitignore # Файл для исключения временных файлов
└── README.md # Этот файл

Как собрать и запустить

Шаг 1: Сгенерировать тестовые матрицы

g++ generate.cpp -o generate.exe
./generate.exe
Шаг 2: Скомпилировать основную программу

g++ -fopenmp -O2 main.cpp -o main.exe
Шаг 3: Запустить эксперименты

./main.exe
Шаг 4: Сохранить результаты в файл

./main.exe > results.txt
Параметры экспериментов
Размеры матриц: 200, 400, 800, 1200, 1600, 2000

Количество потоков: 1, 2, 4, 8

Выводы
Ускорение на 8 потоках достигает примерно 5.6 раз

Эффективность параллелизации растет с увеличением размера матрицы

Рекомендуемое количество потоков = количеству физических ядер процессора
